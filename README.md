# Bookmarklets

JS/HTML bookmarklets

In your browser:
1. Add new bookmark
2. paste file-content/first line as url (view a bookmarklet and use the "copy raw content"-button)
3. chose name to your liking and save
4. click bookmark


## Fullscreen

Click anywhere in the canvas to go fullscreen

### [whiterabbit.bookmarklet](https://github.com/000MDK/Bookmarklets/Fullscreen/whiterabbit.bookmarklet)
